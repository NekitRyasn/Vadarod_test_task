Описание:

В коде используются две модели
1) SamLowe/roberta-base-go_emotions
Содержит 58,000 комментариев с метками эмоций и включает 28 эмоцию в которых есть необходимые по ТЗ.
Текст разбивается на токены, которые модель может понимать.
Я пробовал различные модели, например michellejieli/emotion_text_classifier либо dima806/text-emotion-classifier-distilbert,
но они менее точно определяли эмоции и не имели некоторых, нужных по ТЗ

2)spacy/en_core_web_sm
SpaCy — библиотека, которая предоставляет инструменты для обработки текстов: токенизацию, определение частей речи, извлечение именованных сущностей
Модель en_core_web_sm определяет грамматические связи между словами, что позволяет извлекать корневые глаголы и существительные, что используется для анализа причин эмоций (злости)
